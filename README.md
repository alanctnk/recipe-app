# App de Receitas
### Projeto feito consumido duas API's:
- [The Meal DB](https://themealdb.com/)
- [The Cocktail DB](https://thecocktaildb.com/)

## Tecnologias utilizadas:
- ReactJS
- React Router
- Bootstrap
- Redux

---
## Instalação:
No terminal: `npm install`, após `npm start`.
